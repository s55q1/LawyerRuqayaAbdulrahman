# دليل تثبيت وتشغيل النظام

## المتطلبات
- Node.js 18+ (https://nodejs.org)
- npm أو yarn

## خطوات التثبيت

### 1. تثبيت الحزم
```bash
npm install
```

### 2. إعداد قاعدة البيانات
```bash
npm run db:generate
npm run db:push
```

### 3. إضافة البيانات الأولية (مستخدمين افتراضيين)
```bash
npm run db:seed
```

### 4. تشغيل المشروع
```bash
npm run dev
```

ثم افتح المتصفح على: http://localhost:3000

## روابط مهمة
- الموقع التعريفي: http://localhost:3000
- لوحة التحكم: http://localhost:3000/dashboard
- تسجيل الدخول: http://localhost:3000/auth/login
- Prisma Studio (إدارة قاعدة البيانات): `npm run db:studio`

```
├── schema.prisma           # مخطط قاعدة البيانات

```

## النسخ الاحتياطي
- انسخ ملف `prisma/dev.db` بانتظام (يومياً أو أسبوعياً)
- هذا الملف يحتوي على جميع بيانات النظام

## للإنتاج (Production)
1. غيّر `AUTH_SECRET` في ملف `.env.local` إلى مفتاح عشوائي قوي
2. انتقل إلى قاعدة بيانات PostgreSQL بدلاً من SQLite
3. فعّل HTTPS
